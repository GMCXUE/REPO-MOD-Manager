# REPOKit — REPO MOD 管理器

> 专为游戏 **R.E.P.O.** 设计的 MOD 管理工具，支持一键安装 BepInEx、在线浏览/安装 Thunderstore MOD、管理本地已安装 MOD。

---

## 功能概览

| 功能 | 说明 |
|------|------|
| 🎮 启动游戏 | 一键通过 Steam 启动 R.E.P.O. |
| ⬇ 安装 BepInEx | 自动从 Thunderstore 下载并安装 BepInEx MOD 框架 |
| 🌐 在线浏览 MOD | 浏览 Thunderstore 上的全部 REPO MOD，支持搜索、排序、分页 |
| 🇨🇳 中文 MOD 筛选 | 一键筛选中文汉化 MOD |
| 📦 本地 MOD 管理 | 卡片样式展示已安装 MOD，支持图标预览和一键删除 |
| 📥 ZIP 导入安装 | 直接导入本地 ZIP 包安装 MOD |
| 🔄 自动扫描路径 | 启动时自动读取 Steam 库路径，免手动配置 |

---

## 使用前提

- **Windows 10/11**（64位）
- 已安装 **Steam** 并购买 **R.E.P.O.**
- 无需安装 Python（直接运行 `.exe`）

---

## 快速开始

### 第一次使用（全新安装）

1. 从 [Releases](https://github.com/GMCXUE/REPO-MOD-Manager/releases) 下载最新版 `REPO_MOD_Manager.exe`
2. 双击运行，软件会**自动扫描**游戏和 plugins 目录
3. 若未自动识别，点击「浏览…」手动选择对应目录：
   - **游戏目录**：含 `REPO.exe` 的文件夹，例如 `D:\SteamLibrary\steamapps\common\REPO`
   - **plugins 目录**：`游戏目录\BepInEx\plugins`
4. 如果还没有安装 BepInEx，点击「**⬇ 安装 BepInEx**」一键安装
5. 安装完 BepInEx 后，先**运行一次游戏**再关闭，让 BepInEx 完成初始化
6. 之后即可正常安装 MOD

### 安装 MOD

**方式一：在线安装（推荐）**
1. 切换到「🌐 在线浏览安装」标签页
2. 等待 MOD 列表加载完成（首次约 10 秒）
3. 使用搜索框搜索，或点击「🇨🇳 中文 MOD」筛选汉化包
4. 点击 MOD 卡片查看详情，点击「⬇ 下载并安装」
5. 安装完成后自动刷新本地列表

**方式二：导入 ZIP 安装**
1. 切换到「📂 已安装 MOD」标签页
2. 点击「📦 导入 ZIP 安装」，选择本地 ZIP 文件
3. 软件自动解压到正确目录

### 删除 MOD

在「📂 已安装 MOD」标签页，点击对应 MOD 卡片上的「🗑 删除」按钮。

## 配置文件

配置自动保存在：
```
%APPDATA%\REPOModManager\config.json
```
内容包括游戏目录和 plugins 目录路径，重装软件后路径不丢失。

## 常见问题

**Q：软件启动后没有自动识别路径？**
A：点击「浏览…」手动选择。游戏目录通常在 `C:\Program Files (x86)\Steam\steamapps\common\REPO` 或其他 Steam 库盘符下。

**Q：安装 MOD 后游戏没有加载？**
A：确认已安装 BepInEx，并且游戏至少运行过一次。

**Q：安装 BepInEx 失败？**
A：检查网络连接是否正常（Thunderstore 国内可直连）。确认游戏目录设置正确。

**Q：翻译功能无法使用？**
A：翻译功能使用 Google Translate，在中国大陆需要 VPN。MOD 下载和安装功能不需要 VPN。

**Q：启动游戏按钮没有反应？**
A：确认已设置正确的游戏目录，且 Steam 正在运行。
